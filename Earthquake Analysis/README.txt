Order of Executing the code:

1. Run quakes-clean = this will create the CSV file 

2. Run quakes-insights = this will show the 5 insights

3. Run quakes-plots = this will show the visualization of the 5 insights which was shown in the quakes-insights

Before running quakes-shiny make sure you installed kaleido - pip install -U kaleido ( if not use this command)

4. Run quakes-shiny = this create a local app which can be viewed in your web-browser
